🚀 Quick Install

  1. Open Eclipse IDE
  2. Go to Help → Install New Software...
  3. Click Add... to add a new update site
  4. Enter a name (e.g., "Sandbox") and paste one of the update site URLs above
  5. Select the cleanup features you want to install
  6. Follow the installation wizard and accept the license
  7. Restart Eclipse when prompted
