What is Gemini Mining?

  • Automated Commit Analysis: Uses Google's Gemini AI to analyze commit patterns
  • Refactoring Opportunities: Identifies potential code improvements and cleanup opportunities
  • DSL Rule Proposals: Suggests new cleanup rules based on discovered patterns
  • Traffic Light System: Evaluates proposals with GREEN/YELLOW/RED ratings

How It Works

  • The gemini-mining.yml workflow runs nightly
  • Analyzes recent commits from configured repositories
  • Gemini AI evaluates each commit for refactoring patterns
  • Results are compiled into this dashboard
  • Proposals can be applied via apply-mining-proposal.yml workflow
